Команды

Перечень команд, их наборов и биндов с их описанием.

Этот список ориентирован только на комманды которые нужно использовать в

локальном сервере. Перед всем включите читы при помощи "sv_cheats true".

Бінди:

alias +pickup "ent_grab"; або -pickup "ent_ungrab"; bind c "+pickup" - Підбір

предметів, затискаючи кнопку 'c'. (Як у Half Life 2)

bind mwheeldown "+jump" - Стрибок на коліщатко мишки вниз.

bind v "noclip" - режим польоту на кнопку 'v'.

bind ] "bot_place" - При натисканні ']' бот буде телепортований до вашого прицілу.

Команда для тренування:

sv_infinite_ammo true - Безкінечні патрони.

mp_warmup_pausetimer true; mp_warmup_start - Увімкнути нескінченну розминку.

sv_showimpacts true - Показати влучення кулі для клієнта та сервера.

sv_grenade_trajectory_prac_pipreview true - Відобразити камеру стеження за гранатами.

mp_buytime 99999999; mp_buy_anywhere true; mp_restartgame true - Дозволити покупки скрізь та постійно.

mp_freezetime 0 - Забрати заморозку на початку раунду.

mp_warmup_end - Закінчити розминку mp_autoteambalance false - Вимкнути баланс гравців у командах.

sv_enablebunnyhopping true; sv_autobunnyhopping true; - Увімкнути авто-розстрибування на сервері.

Боти:

bot_add – Додати бота.

bot_add_ct – Додати бота за кт.

bot_add_t - Додати бота за т.п.

bot_kick - Видалити всі роботи;

bot_kill - Вбити всіх роботів.

bot_stop true - Зупинити всі роботи.

bot_mimic true - Змусити всіх роботів повторювати ваші дії.

bot_mimic_yaw_offset 0 - Розгорнути ботів від вас (для попередньої команди).

bot_crouch true - Змусити всіх ботів сісти.

bot_place - Телепортувати бота до вашого прицілу.

bot_ignore_players true – Боти вас ігнорують.

bot_loadout ak47 - Усі боти купуватимуть AK47 (навіть кт).

Корисне:

ent_remove_all planted_c4 - Видалити всі встановлені C4.

ent_teleport hostage_entity - Телепортувати заручника на місце прицілу.

mat_fullbright true - Вимкнути всі тіні та освітлення.

cl_inventory_debug_tooltip true - Відображати додаткову інформацію про

предмети в інвентарі.

map_showbombradius - Увімкнути відображення радіусу вибуху бомби. (Вимикається саме)

sv_disable_teamselect_menu true - Вимкнути зміну сторін на сервері.

sv_disable_radar true - Вимкнути радар для всіх на сервері.

Весілля:

ent_fire weapon_* changesubclass weapon_ak47 - Замінити моделі всіх зброї на калаші.

ent_create planted_c4 - Створити встановлену С4 на місці прицілу.

sv_jump_impulse 600 – Високі стрибки для всіх на сервері.

sv_gravity 300 - Гравітація як у перелітних снайперах.

host_timescale 0.2 - Уповільнений час.

host_timescale 1 - Нормальний перебіг часу.

host_timescale 5 - Прискорений час.
